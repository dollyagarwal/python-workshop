# Exercises: Data Handling & I/O

1. **Strings**  
   Normalize whitespace in a sentence and title-case names.

2. **Lists/Dicts/Sets**  
   Deduplicate a list of email addresses and count occurrences.

3. **JSON**  
   Parse JSON from a string representing a list of users; filter by active status.

4. **File I/O**  
   Write filtered users to `active_users.json`.

5. **API Call (mock)**  
   Implement a function `fetch_posts()` that returns sample data (simulate API).

*Stretch:* Convert the result to a `pandas.DataFrame` and compute simple stats.
