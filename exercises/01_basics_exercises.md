# Exercises: Basics & Syntax

1. **Variables & Types**  
   Create variables of different types (int, float, str, list, dict). Print their types.

2. **Control Flow**  
   Write a function `grade(score)` that returns 'A', 'B', 'C', 'D', or 'F'.

3. **Functions**  
   Implement `fizzbuzz(n=20)` that prints numbers 1..n, with Fizz/Buzz rules.

4. **Default & Keyword Args**  
   Write `def repeat(s, times=2, sep='-')` and test with positional and keyword args.

5. **Error Handling**  
   Implement `safe_div(a, b)` returning `None` if division by zero occurs, otherwise the result.

*Stretch:* Show how dynamic typing affects `safe_div` (e.g., string inputs) and add type hints.
